# CountSemiprimes — enhanced

This repository contains small Python examples and notes related to semiprimes and related algorithmic tasks.

What was changed in this enhancement:
- Added a math/ folder with LaTeX-formatted Markdown notes for CountSemi and ChocolatesByNumbers.
- Added SUPER_DISCOVERY.md summarizing recent mathematical developments and trends.
- Added module docstrings to Python files pointing to math/ notes.
- Added this README with usage examples.

Usage

- To run examples:
    python CountSemi.py
    python ChocolatesByNumbers.py

- Math notes are in the math/ folder (CountSemi.md, ChocolatesByNumbers.md).

Contributing

Please open issues or PRs with improvements. The math notes are intended as companion documentation and can be expanded with citations.
